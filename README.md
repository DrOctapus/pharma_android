# Pharma Android

Final Project for Software Engineering Course, AUEB 2024. An Android Pharmaceutical application designed to manage prescriptions, active substances, and pharmaceutical products.

## Prerequisites
* **Android Studio**: Hedgehog (2023.1.1) or newer.
* **Java Development Kit (JDK)**: Version **11** is required.
* **Android SDK**:
    * **Compile SDK**: 34
    * **Minimum SDK**: 24

## Build
Open the project in Android Studio.

Click Make Project (Ctrl+F9).

Select a device/emulator and click Run.
